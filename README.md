<h1><strong>Zoo Functions</strong></h1>

<h2><strong>Descrição</strong></h2>
<p>
  O objetivo deste projeto é manipular e consultar dados de um zoológico municipal utilizando JavaScript ES6 e Higher Order Functions.  
  As funções implementadas permitem buscar informações sobre espécies, locais de origem dos animais e dados das pessoas responsáveis pelos cuidados, sempre com foco em código limpo, modular e testado com Jest.
</p>

<h2><strong>Funcionalidades</strong></h2>
<ul>
  <li>
    <strong>Busca de informações dos animais.</strong>  
    Consulta de dados como espécie, local de origem, idade e outras características relevantes.
  </li>
  <li>
    <strong>Consulta de informações dos cuidadores.</strong>  
    Acesso a dados das pessoas que trabalham no zoológico, incluindo nome, cargo e áreas de atuação.
  </li>
  <li>
    <strong>Uso de ES6 e Higher Order Functions.</strong>  
    Implementação de funções com recursos modernos do JavaScript, como arrow functions, array methods, objetos e desestruturação.
  </li>
  <li>
    <strong>Testes automatizados com Jest.</strong>  
    Criação de testes unitários para garantir o correto funcionamento de cada função e facilitar a manutenção do código.
  </li>
</ul>

<h2><strong>Demonstração do Projeto</strong></h2>
<p>
  Este projeto é focado em lógica de programação e manipulação de dados, portanto não possui uma interface visual em produção.  
  A execução das funções e dos testes pode ser feita diretamente no ambiente de desenvolvimento com <strong>Node.js</strong> e <strong>Jest</strong>.
</p>

  <h2><strong>Description</strong></h2>
  <p> 
    The goal of this project is to work with JavaScript ES6 and Higher Order Functions to query and manipulate data from a municipal zoo.

    </li>
    <li>
      <strong>Staff information queries.</strong>  
      Functions to retrieve data about the zoo staff, including name, role and areas of expertise.
    </li>
    <li>
      <strong>ES6 and Higher Order Functions.</strong>  
      Use of modern JavaScript features and HOFs to keep the code modular, readable and reusable.
    </li>
    <li>
      <strong>Automated tests with Jest.</strong>  
      Unit tests for each function, ensuring correctness and easier maintenance.
    </li>
